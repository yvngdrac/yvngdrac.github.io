# My Writeups Portfolio

A collection of my security research, CTF writeups, and technical posts.

## Quick Start

1. Clone this repo: `git clone https://github.com/yvngdrac/yvngdrac.github.io.git`
2. Create new posts in the `_posts/` folder
3. Name your files: `YYYY-MM-DD-your-title.md`
4. Push to GitHub
5. Your site updates automatically at https://yvngdrac.github.io

## Adding Writeups

Each post needs a header (frontmatter):

```markdown
---
title: "Your Writeup Title"
date: 2026-05-18
categories: [writeups]
---

# Your Title

Your content here...
```

## Customization

Edit `_config.yml` to:
- Change site title
- Update author name
- Choose a different theme
- Add your email
